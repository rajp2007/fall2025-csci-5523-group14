# CSCI 5523 Data Mining - Group 14 Project

**Fall 2025 | University of Minnesota**

## Project Overview

This repository contains the academic group project for CSCI 5523: Introduction to Data Mining. Our team explores developing a machine learning model to predict the likelihood of a newborn experiencing an adverse event or illness using the "Newborn Health Monitoring Dataset" from Kaggle.
